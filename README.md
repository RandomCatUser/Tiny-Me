# Tiny Me 

**Tiny Me** is an elegant, lightweight AI chatbot built with **PyQt5** and **HTML** that runs locally with a smooth, PurrGPT-inspired interface.  
It uses the **Pollinations AI API** (`https://text.pollinations.ai/{prompt}`) for generating intelligent responses.

---

## Features

-  **PurrGPT-Inspired UI** — minimal, modern, and animated design  
-  **Smart Chat** — communicate with AI using Pollinations’ text API  
-  **Local Memory** — chat history is stored and reloaded via `localStorage`  
-  **Hidden Sidebar** — access settings, clear memory, and prompt options  
-  **PyQt5 Engine** — HTML runs seamlessly in a native desktop window  
-  **Animated Replies** — typing effect similar to GPT chat  
-  **Persistent Conversations** — keeps memory between sessions

---

## Tech Stack

| Component | Description |
|------------|-------------|
| **Frontend** | HTML, CSS, JavaScript |
| **Backend** | PyQt5 (Python GUI Engine) |
| **AI Engine** | [Pollinations Text API](https://text.pollinations.ai) |
| **Storage** | localStorage for saving memory |
