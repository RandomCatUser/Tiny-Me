# Tiny Me 

**Tiny Me** is an elegant, lightweight AI chatbot built with **PyQt5** and **HTML** that runs locally with a smooth, PurrGPT-inspired interface.  
It uses the **Pollinations AI API** (`https://text.pollinations.ai/{prompt}`) for generating intelligent responses.
